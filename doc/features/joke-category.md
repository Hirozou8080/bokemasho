# ボケカテゴリ

## 概要
ボケ投稿時にカテゴリ（タグ）を設定できる機能。ユーザーが自由にタグを入力でき、既存のタグはサジェストされる。

## 仕様

### カテゴリの設定
- 1つのボケに対して最大3つまでカテゴリを設定可能
- カテゴリ名は最大6文字
- 自由入力形式（既存のカテゴリから選択 or 新規作成）
- 同じボケに同じカテゴリの重複設定は不可

### カテゴリ入力UI
- タップで追加するボタン形式（モバイルフレンドリー）
- 入力中のタグをChipで表示し、タップで追加
- 既存カテゴリのサジェスト表示（最大6件）
- 選択済みカテゴリはChipで表示、削除可能
- Enterキーでのフォーム送信を防止

### カテゴリ表示
- ボケ一覧（ホーム画面、ボケ一覧画面）でカテゴリをChipで表示
- カテゴリがない場合も表示位置は固定（レイアウト崩れ防止）

## 関連API

### カテゴリ一覧取得
```
GET /api/categories?search={keyword}
```
- サジェスト用のカテゴリ一覧を取得
- searchパラメータで部分一致検索

### ボケ投稿（カテゴリ付き）
```
POST /api/jokes/create
```
**リクエスト:**
```json
{
  "topic_id": 1,
  "content": "ボケの内容",
  "categories": ["シュール", "天然ボケ"]
}
```
- 新しいカテゴリ名は自動的に作成される
- categoriesは最大3要素、各要素は最大6文字

### ボケ取得レスポンス
```json
{
  "id": 1,
  "content": "ボケの内容",
  "categories": [
    { "id": 1, "name": "シュール" },
    { "id": 2, "name": "天然ボケ" }
  ],
  ...
}
```

## 関連テーブル
- `categories` - カテゴリマスタ
- `joke_category` - ボケとカテゴリの中間テーブル（多対多）

## 関連画面
- ボケ投稿画面: `/joke_topic/[id]`
- ホーム画面: `/`
- ボケ一覧画面: `/jokes`
