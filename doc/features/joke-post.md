# ボケ投稿

## 概要
ボケの投稿を行うことができる機能。
ログイン済みのユーザしかボケの投稿はできない。

## 投稿フロー
1. お題一覧から投稿したいお題を選択
2. お題の画像を見ながらボケを入力（最大60文字、改行不可）
3. カテゴリ（タグ）を設定（任意、最大3つ、各6文字まで）
4. 「ボケを投稿」ボタンで投稿
5. 成功後、ボケ一覧画面に遷移

## 仕様

### ボケ入力
- 最大60文字
- 改行は自動的に削除される
- 文字数カウンター表示（例: 「30/60文字」）

### カテゴリ設定
- 任意（設定しなくても投稿可能）
- 最大3つまで設定可能
- 各カテゴリ名は最大6文字
- 詳細は [ボケカテゴリ](./joke-category.md) 参照

## 関連API
```
POST /api/jokes/create
```
**リクエスト:**
```json
{
  "topic_id": 1,
  "content": "ボケの内容",
  "categories": ["タグ1", "タグ2"]
}
```

## 関連画面
- ボケ投稿画面: `/joke_topic/[id]`
